# rreading-glasses

Metadata service for book and audiobook management applications

## Introduction

This chart bootstraps a [rreading-glasses](https://github.com/geekxflood/helm-charts/tree/main/charts/rreading-glasses) deployment on a [Kubernetes](http://kubernetes.io) cluster using the [Helm](https://helm.sh) package manager.

## Installation

```bash
helm install rreading-glasses ./charts/rreading-glasses
```
