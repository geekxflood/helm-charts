# dizquetv

A Helm chart for Kubernetes for deploying DizqueTV

## Introduction

This chart bootstraps a [dizquetv](https://github.com/geekxflood/helm-charts/tree/main/charts/dizquetv) deployment on a [Kubernetes](http://kubernetes.io) cluster using the [Helm](https://helm.sh) package manager.

## Installation

```bash
helm install dizquetv ./charts/dizquetv
```
