# flaresolverr

A Helm chart for Kubernetes for deploying Flaresolverr

## Introduction

This chart bootstraps a [flaresolverr](https://github.com/geekxflood/helm-charts/tree/main/charts/flaresolverr) deployment on a [Kubernetes](http://kubernetes.io) cluster using the [Helm](https://helm.sh) package manager.

## Installation

```bash
helm install flaresolverr ./charts/flaresolverr
```
