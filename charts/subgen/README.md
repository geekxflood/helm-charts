# subgen

SubGen - Autogenerate subtitles using OpenAI Whisper Model

## Introduction

This chart bootstraps a [subgen](https://github.com/geekxflood/helm-charts/tree/main/charts/subgen) deployment on a [Kubernetes](http://kubernetes.io) cluster using the [Helm](https://helm.sh) package manager.

## Installation

```bash
helm install subgen ./charts/subgen
```
