# tdarr

Tdarr Helm chart for Kubernetes - Distributed transcode system for automating media library transcode/remux management

## Introduction

This chart bootstraps a [tdarr](https://github.com/geekxflood/helm-charts/tree/main/charts/tdarr) deployment on a [Kubernetes](http://kubernetes.io) cluster using the [Helm](https://helm.sh) package manager.

## Installation

```bash
helm install tdarr ./charts/tdarr
```
