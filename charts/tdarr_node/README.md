# tdarr-node

A Helm chart for Kubernetes for deploying tdarr transcoding nodes

## Introduction

This chart bootstraps a [tdarr-node](https://github.com/geekxflood/helm-charts/tree/main/charts/tdarr_node) deployment on a [Kubernetes](http://kubernetes.io) cluster using the [Helm](https://helm.sh) package manager.

## Installation

```bash
helm install tdarr-node ./charts/tdarr_node
```
