# tdarr-server

A Helm chart for Kubernetes for deploying tdarr server

## Introduction

This chart bootstraps a [tdarr-server](https://github.com/geekxflood/helm-charts/tree/main/charts/tdarr_server) deployment on a [Kubernetes](http://kubernetes.io) cluster using the [Helm](https://helm.sh) package manager.

## Installation

```bash
helm install tdarr-server ./charts/tdarr_server
```
