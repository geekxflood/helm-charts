# oauth2-proxy

OAuth2 Proxy for authenticating applications via Keycloak OIDC

## Introduction

This chart bootstraps a [oauth2-proxy](https://github.com/geekxflood/helm-charts/tree/main/charts/oauth2-proxy) deployment on a [Kubernetes](http://kubernetes.io) cluster using the [Helm](https://helm.sh) package manager.

## Installation

```bash
helm install oauth2-proxy ./charts/oauth2-proxy
```
