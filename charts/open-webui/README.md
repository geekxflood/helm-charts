# open-webui

Open WebUI - User-friendly web interface for LLMs like Ollama

## Introduction

This chart bootstraps a [open-webui](https://github.com/geekxflood/helm-charts/tree/main/charts/open-webui) deployment on a [Kubernetes](http://kubernetes.io) cluster using the [Helm](https://helm.sh) package manager.

## Installation

```bash
helm install open-webui ./charts/open-webui
```
