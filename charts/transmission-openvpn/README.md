# transmission-openvpn

A Helm chart for Kubernetes for deploying Transmission

## Introduction

This chart bootstraps a [transmission-openvpn](https://github.com/geekxflood/helm-charts/tree/main/charts/transmission-openvpn) deployment on a [Kubernetes](http://kubernetes.io) cluster using the [Helm](https://helm.sh) package manager.

## Installation

```bash
helm install transmission-openvpn ./charts/transmission-openvpn
```
