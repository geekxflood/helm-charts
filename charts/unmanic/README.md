# unmanic

A Helm chart for Kubernetes for deploying Unmanic

## Introduction

This chart bootstraps a [unmanic](https://github.com/geekxflood/helm-charts/tree/main/charts/unmanic) deployment on a [Kubernetes](http://kubernetes.io) cluster using the [Helm](https://helm.sh) package manager.

## Installation

```bash
helm install unmanic ./charts/unmanic
```
