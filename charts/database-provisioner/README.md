# database-provisioner

Automated database provisioner for CloudNativePG Database CRDs

## Introduction

This chart bootstraps a [database-provisioner](https://github.com/geekxflood/helm-charts/tree/main/charts/database-provisioner) deployment on a [Kubernetes](http://kubernetes.io) cluster using the [Helm](https://helm.sh) package manager.

## Installation

```bash
helm install database-provisioner ./charts/database-provisioner
```
