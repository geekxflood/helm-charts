# tautulli-exporter

A Helm chart for Kubernetes for deploying Tautulli-exporter

## Introduction

This chart bootstraps a [tautulli-exporter](https://github.com/geekxflood/helm-charts/tree/main/charts/tautulli-exporter) deployment on a [Kubernetes](http://kubernetes.io) cluster using the [Helm](https://helm.sh) package manager.

## Installation

```bash
helm install tautulli-exporter ./charts/tautulli-exporter
```
