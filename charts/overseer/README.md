# overseer

A Helm chart for Kubernetes

## Introduction

This chart bootstraps a [overseer](https://github.com/geekxflood/helm-charts/tree/main/charts/overseer) deployment on a [Kubernetes](http://kubernetes.io) cluster using the [Helm](https://helm.sh) package manager.

## Installation

```bash
helm install overseer ./charts/overseer
```
