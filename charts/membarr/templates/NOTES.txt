Membarr has been deployed!

Membarr is a Discord bot for managing Plex/Jellyfin user invitations.

1. Check the pod status:
   kubectl get pods --namespace {{ .Release.Namespace }} -l "app.kubernetes.io/name={{ include "membarr.name" . }},app.kubernetes.io/instance={{ .Release.Name }}"

2. View the logs:
   kubectl logs --namespace {{ .Release.Namespace }} -l "app.kubernetes.io/name={{ include "membarr.name" . }},app.kubernetes.io/instance={{ .Release.Name }}"

3. Configuration:
   - Config files are stored in: /app/app/config
   {{- if .Values.persistence.enabled }}
   - Persistence is enabled with a {{ .Values.persistence.size }} volume
   {{- else }}
   - WARNING: Persistence is disabled! Config will be lost on pod restart.
   {{- end }}

4. Discord Setup:
   After the bot is online, use these commands in Discord to configure:

   For Jellyfin:
   /jellyfinsettings setup <server url> <api key>
   /jellyfinsettings addrole <@role>
   /jellyfinsettings enable

   For Plex:
   /plexsettings setup <username> <password> <server name>
   /plexsettings addrole <@role>
   /plexsettings enable

For more information, visit: https://github.com/geekxflood/Membarr
