# tunarr

A Helm chart for Tunarr - Create live TV channels from media on your Plex/Jellyfin/Emby servers with built-in HDHomeRun emulation

## Introduction

This chart bootstraps a [tunarr](https://github.com/geekxflood/helm-charts/tree/main/charts/tunarr) deployment on a [Kubernetes](http://kubernetes.io) cluster using the [Helm](https://helm.sh) package manager.

## Installation

```bash
helm install tunarr ./charts/tunarr
```
