# tautulli

A Helm chart for Kubernetes for deploying Tautulli

## Introduction

This chart bootstraps a [tautulli](https://github.com/geekxflood/helm-charts/tree/main/charts/tautulli) deployment on a [Kubernetes](http://kubernetes.io) cluster using the [Helm](https://helm.sh) package manager.

## Installation

```bash
helm install tautulli ./charts/tautulli
```
