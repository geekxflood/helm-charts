# wizarr

A Helm chart for deploying Wizarr - automatic user invitation and management system for media servers

## Introduction

This chart bootstraps a [wizarr](https://github.com/geekxflood/helm-charts/tree/main/charts/wizarr) deployment on a [Kubernetes](http://kubernetes.io) cluster using the [Helm](https://helm.sh) package manager.

## Installation

```bash
helm install wizarr ./charts/wizarr
```
