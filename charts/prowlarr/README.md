# prowlarr

A Helm chart for Kubernetes

## Introduction

This chart bootstraps a [prowlarr](https://github.com/geekxflood/helm-charts/tree/main/charts/prowlarr) deployment on a [Kubernetes](http://kubernetes.io) cluster using the [Helm](https://helm.sh) package manager.

## Installation

```bash
helm install prowlarr ./charts/prowlarr
```
