# overseerr

A Helm chart for Kubernetes for deploying Overseerr

## Introduction

This chart bootstraps a [overseerr](https://github.com/geekxflood/helm-charts/tree/main/charts/overseerr) deployment on a [Kubernetes](http://kubernetes.io) cluster using the [Helm](https://helm.sh) package manager.

## Installation

```bash
helm install overseerr ./charts/overseerr
```
