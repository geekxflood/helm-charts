# readarr

A Helm chart for deploying Readarr on Kubernetes

## Introduction

This chart bootstraps a [readarr](https://github.com/geekxflood/helm-charts/tree/main/charts/readarr) deployment on a [Kubernetes](http://kubernetes.io) cluster using the [Helm](https://helm.sh) package manager.

## Installation

```bash
helm install readarr ./charts/readarr
```
