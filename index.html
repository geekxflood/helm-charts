Charts for self-hosters who run real clusters.

45 curated Helm charts — media servers, the *arr stack, downloaders, AI/LLM, identity, and the infra that ties them together. Opinionated defaults, Gateway API support, GPU-ready where it matters.

Browse charts View on GitHub

Add the repo

helm repo add geekxflood https://geekxflood.github.io/helm-charts Copy

Then helm repo update and install any chart with helm install <name> geekxflood/<chart>.
